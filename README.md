# SystemForTeam: Sistema de Gerenciamento de Projetos Empresariais

Olá, bem-vindo ao SystemForTeam! Este é um sistema de gestão empresarial desenvolvido pelos alunos **Moisés, Yohan, Igor e Mark** para a disciplina de Desenvolvimento de Sistemas.

### Descrição

O **SystemForTeam** é uma solução robusta para otimizar o gerenciamento de projetos empresariais. Com uma interface intuitiva e funcionalidades abrangentes, ele facilita o planejamento, a execução e o monitoramento de projetos, promovendo a colaboração entre equipes e aumentando a eficiência operacional.

### Funcionalidades Principais

- **Planejamento de Projetos**: Ferramentas para definição de metas, cronogramas e alocação de recursos de forma eficiente.
- **Colaboração em Tempo Real**: Permite que equipes compartilhem documentos, comentários e atualizações instantaneamente, facilitando a comunicação.
- **Gestão de Tarefas**: Atribuição de tarefas a membros da equipe, com prazos e prioridades bem definidas, ajudando a manter o foco nas entregas.
- **Monitoramento e Relatórios**: Painéis de controle que permitem o acompanhamento do progresso dos projetos, além de relatórios detalhados sobre desempenho, tempo e recursos utilizados.
- **Integração com Ferramentas de Terceiros**: Possibilidade de integração com plataformas como Slack, Google Drive e outros serviços, potencializando a funcionalidade do sistema.
- **Análise de Risco**: Ferramentas que ajudam a identificar e mitigar riscos, assegurando que os projetos sejam concluídos com sucesso.

### Instalação e Uso  
**Pré-Requisitos:** Python e utilização de Powershell, Terminal SSH ou Console.

**Passos para configuração:**
1. **Renomeie o arquivo** `enviroment.env` **para** `.env`.
2. **Configure o banco de dados**: Execute o XAMPP ou outro serviço MySQL com a porta 3307, sem senha, e com a database **system4team**. Esta configuração pode ser alterada dentro do arquivo `.env`, se necessário.
